## pre-emptive OS
  - multitasking OS
  - suspend currently running process
  - determine which process should execute next
---

## process scheduling
  - priority algorithm
    : the higher priority, the earlier running
  - round-robin algorithm
    - if same priority
      - time slice
---
## scheduler
  - when?
    - every time slice
    - a process created or a process ended
    - a running process blocked
