## set 
```python
s = set([1, 2, 3, 4])

3. add() : 한 개 추가
>>>s.add(5) 
>>>s
{1, 2, 3, 4, 5}

4. update() : 여러 개 추가
>>>s.add([6, 7, 8])
>>>s
{1, 2, 3, 4, 5, 6, 7, 8}

5. remove() : 한 개 제거
>>s.remove(5)
>>>s
>>>{1, 2, 3, 4, 6, 7, 8}
```

---
## set 
```python
s1 = set([1, 2, 3, 4])
s2 = set([3, 4, 5, 6])

6. & or intersection() : 교집합
>>>s1 & s2
{3, 4}
>>>s1.intersection(s2)

7. | or union() : 합집합
>>>s1 | s2
{1, 2, 3, 4, 5, 6}
>>>s1.union(s2)

8. - or difference() : 차집합
>>>s1 - s2
{1, 2}
>>>s1.difference(s2)
```

## Leap year
4로 나뉘어 떨어지면 윤년,
100으로 나뉘어 떨어지면 평년,
400으로 나뉘어 떨어질땐 윤년

---
## Leap year(answer)
```python

def is_leap(y):
	leap = False
	if y % 4 == 0 and (y % 100 != 0 or y % 400 == 0)
		leap = True
	return leap
    
y = int(input("Is leap?? "))
print(is_leap(y))

```
